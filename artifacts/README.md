# APSS artifacts

This directory contains outputs produced for consumers by the APSS Framework
Stewardship System.

- [framework/](framework/) is the primary, normative artifact. Its definition,
  schema, template, visualization contract, and changelog collectively specify
  APSS.
- [examples/](examples/) contains supporting, non-normative artifacts that
  demonstrate implementations and test the framework against concrete domains.

Operational processes, work state, evidence handling, validation procedures,
and compiled stewardship knowledge live under [`operations/`](../operations/).
