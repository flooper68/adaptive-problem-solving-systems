# Brainstorming

Use this work session when a production idea, task, unresolved failure, or
research topic needs iterative discussion with the responsible user.

1. Frame the topic, affected goal and open problem when applicable, production
   outcome, relevant evidence, and useful stopping point.
2. Read the current CNC system declaration, strategy, problem and task files,
   manufacturing knowledge, relevant sessions, inspection evidence, assembly feedback,
   and applicable references.
3. Discuss observations, alternatives, constraints, and trade-offs with the
   responsible user, asking one load-bearing question at a time.
4. Compile the current conclusion directly into reviewable, uncommitted changes
   to this instantiation or `knowledge/README.md`. Update
   `knowledge/CHANGELOG.md` when compiled manufacturing knowledge changes.
5. Present the changes to the user and iterate until they accept the result,
   explicitly stop, or retain a clear unresolved question or next trigger.
6. Do not change approved tooling, validation, or operating limits without the
   manufacturing lead's declared adaptation authority.
