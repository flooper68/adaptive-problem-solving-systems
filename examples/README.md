# APSS implementation examples

These are supporting artifacts produced to help consumers understand APSS and
to expose gaps in the framework through concrete application.

Examples are non-normative. Domain choices, process arrangements, fields, or
techniques used by one example are not APSS requirements unless the normative
[framework definition](../framework/README.md) explicitly says so.

## Available examples

- [CNC part production](cnc-part-production/SYSTEM.md) — a proposed physical
  production system whose selected batch task addresses separate conformance
  and consumer-outcome problems, with brainstorming and problem-grooming work
  sessions plus dimensional and real-use validation processes.
