# Insight stream

This stream retains explicit interpretations derived from working sessions,
feedback, usage, research, validation, work history, and other recoverable
evidence. An insight states what a participant or operator thinks follows from
that evidence. It is not raw observation, established knowledge, an approved
decision, or executable work merely because it is retained.

Use [insight intake](../../processes/insight-intake.md) to preserve and clarify a
claim and [insight grooming](../../processes/insight-grooming.md) to assess its
scope, inference, evidence, confidence, and disposition. Executable responses
and compiled knowledge remain linked but separate artifacts.

This README describes the stream and intentionally does not maintain an
exhaustive index. Discover records by their frontmatter and filenames.
